# Таможенный учёт (customs_control_app)

Мобильное приложение (Flutter) для сотрудника таможни: учёт, контроль и запись
пересекающих границу транспортных средств с фотофиксацией.

## Возможности

- **Запись ТС** с фото (камера/галерея) и автоматическим сжатием для экономии хранилища.
- **Маски госномеров** Узбекистана и Таджикистана с выбором формата и валидацией.
- **Словари марок/моделей** грузовиков (корейские/узбекские и др.) — предзаполнены,
  пополняются «на лету»; компании и типы груза тоже расширяемы.
- **Древовидная группировка**: Компания → Авто / Тип груза.
- **Поиск и сортировка** по компании, марке, госномеру, времени, типу груза.
- **Редактирование с полной историей изменений**; удаление — с предупреждением (soft-delete).
- **Разделы**: Главная (дерево), Журнал, Компании, Статистика (графики), Справочники,
  Экспорт (Excel/PDF), Настройки.
- **Защита входа**: биометрия + PIN, настройка при первом запуске или позже.
- **Локализация**: русский и таджикский.

## Стек

- Flutter + Dart, состояние — **Riverpod**.
- Локальная БД — **Drift (SQLite)**; архитектура с Repository pattern для безболезненного
  переноса на удалённый сервер (REST API) в будущем (заготовка в `lib/data/remote/`).
- `flutter_svg`, `flutter_image_compress`, `image_picker`, `local_auth`,
  `flutter_secure_storage`, `go_router`, `fl_chart`, `excel`, `pdf`, `printing`.

## Структура

```
lib/
  core/       # БД, аутентификация, изображения, роутер, темы, l10n, маски номеров
  data/       # локальные реализации репозиториев + заготовка remote
  domain/     # модели и интерфейсы репозиториев
  features/   # экраны по разделам
assets/
  seed/       # словари марок/моделей и типов груза
  branding/   # иконка и логотип (SVG + сгенерированные PNG)
tool/
  generate_icons.dart  # растеризация SVG → PNG средствами Flutter
```

## Запуск

```bash
flutter pub get
dart run build_runner build --delete-conflicting-outputs   # кодогенерация Drift
flutter run
```

### Иконки приложения

При изменении брендинга:

```bash
flutter test tool/generate_icons.dart   # SVG → PNG
dart run flutter_launcher_icons          # иконки Android/iOS
```

## Тесты

```bash
flutter test
```

Покрыты: валидация масок номеров (UZ/TJ) и логика истории изменений (create/update/delete).
